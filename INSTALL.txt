PRINCIPLES:

1. Build project

2. Copy cdls_ex.exe, cdls_lib.dll to a folder and add to system PATH ("C:\Program Files\cdls")

3. Copy python and powershell wrappers ("C:\Program Files\cdls\wrappers")

4. Add python wrapper path to PYTHONPATH env variable

5. Import powershell wrapper into global powershell profile

6. Undo all these on uninstall
